# Scoring Server Portion

The scoring server is implemented using [Swagger](https://swagger.io/) with [swagger-node](https://github.com/swagger-api/swagger-node) as the CLI.

## Install Dependencies

    npm install

## View API

    npm run edit

## Run Local Server

    npm start
